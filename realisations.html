Réalisations

Application Films Gestion Parc Informatique Préventif Client

Description de l'application:

Application permettant l'inventaire de films avec certaines caractéristiques comme la durée ou la qualité.
Application développée avec HTML5, CSS, le framework AngularJS ainsi que PHP pour la partie serveur.
Les données sont stockées dans une base MYSQL.

Application développée en groupe dans le cadre d'un projet de développement en BTS.
Celle-ci permet de simuler une application de gestion de parc informatique avec gestion des postes et des utilisateurs.
Application développée avec HTML, CSS et PHP. Le javascript est utilisé uniquement pour afficher des pops-up.

Application développée dans le cadre de mon alternance dans l'entreprise Régis Martelet pour faire su suivi client.
Cette application permet selon des critères défini par l'utilisateur permet d'afficher les expéditions pour la période donnée.
Elle permet ensuite de valider certaines expéditions ou d'en reporter le suivi.
Application réalisée en HTML, CSS, jQuery pour la partie javascript et PHP pour la partie serveur.
